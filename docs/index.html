🚀 快速部署

Docker 一键部署

# 拉取最新镜像 docker pull openai118/land-ppt:latest # 运行容器 docker run -d \ --name land-ppt \ -p 8000:8000 \ -e DATABASE_URL="postgresql://user:pass@host:port/db" \ -e API_URL="https://your-api-endpoint.com" \ openai118/land-ppt:latest

Docker Compose 部署

version: '3.8' services: land-ppt: image: openai118/land-ppt:latest ports: - "8000:8000" environment: - DATABASE_URL=postgresql://user:pass@host:port/db - API_URL=https://your-api-endpoint.com - API_ANON_KEY=your-api-key restart: unless-stopped
