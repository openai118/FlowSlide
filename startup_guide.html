🚀 FlowSlide 启动指南

📋 步骤 1: 打开命令行

在项目根目录 E:\pyprojects\try1\flowslide 打开PowerShell或命令提示符

🐍 步骤 2: 激活虚拟环境

.\.venv\Scripts\Activate.ps1

或者使用CMD:

.venv\Scripts\activate.bat

📦 步骤 3: 安装依赖 (如果还没安装)

pip install -r requirements.txt

🚀 步骤 4: 启动服务器

选择以下任一方式:

方式 1: 使用启动脚本 (推荐)

python start_flowslide.py

方式 2: 使用run.py

python run.py

方式 3: 直接使用uvicorn

uvicorn src.flowslide.main:app --host 0.0.0.0 --port 8000 --reload

📍 访问地址

  • 首页(公共): http://localhost:8000/home
  • 主页: http://localhost:8000
  • Web界面(控制台): http://localhost:8000/web
  • API文档: http://localhost:8000/docs
  • 配置页面: http://localhost:8000/web/ai-config

🔧 故障排除

  • 确保Python虚拟环境已激活 (提示符前应有 (.venv))
  • 确保所有依赖已正确安装
  • 检查端口8000是否被其他程序占用
  • 如果遇到模板错误，确保工作目录在项目根目录

💡 快速测试

如果主应用启动有问题，可以先运行简单测试服务器:

python simple_test.py
